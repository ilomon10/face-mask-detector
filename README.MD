# Mask Detection

## Requirement
- Tensorflow Lite
- Adafruit mlx90614
- Flask SocketIO

## Usage
```bash
# -i, --ip      Ip Address (0.0.0.0 = localhost)
# -p, --port    Port
$ python3 stream.py -i 0.0.0.0 -p 3000
```
